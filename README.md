# Vue Web Template

A modern web template built with Nuxt.js and Vue 3. Quickly customize colors, branding, and content to create your own website.

## Features

- **Nuxt 3** + Vue 3
- **Tailwind CSS** for styling
- Easy customization via configuration
- Responsive design
- Pre-built pages (Home, Products, About, Contact, Login)

## Quick Start

```bash
npm install
npm run dev
```

## Customization

All customization is handled in `src/config/site.ts`:

- **Branding**: Logo, name, tagline
- **Colors**: Primary, secondary, accent colors
- **Navigation**: Menu links
- **Footer**: Copyright and links
- **Contact**: Email, phone, address, social links

See [docs/customization.md](docs/customization.md) for detailed instructions.
